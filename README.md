# Agents R Us — Proof Site

Static, pure HTML/CSS/JS. Nine pages. No build step.

## Pages
- `index.html` — Home
- `about.html` — About
- `services.html` — Services
- `how-it-works.html` — How It Works
- `case-studies.html` — Case Studies (illustrative, anonymized)
- `pricing.html` — Pricing (no prices — contact for pricing)
- `blog.html` — Blog (coming soon treatment)
- `contact.html` — Contact (email + form, no phone, no address)
- `book.html` — Book a Consultation

## Local preview
Open `index.html` directly, or run `python3 -m http.server 8000` from this folder.

## Notes
- Email `hello@agentsrus.ai` is a placeholder pending client confirmation.
- Forms are no-backend (preventDefault → confirmation message).
- All imagery is AI-generated. No stock photos of people. No fake team photos.
- Custom SVG wordmark inline in header (also `assets/img/logo.svg`).
